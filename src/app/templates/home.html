Recommendations
A web app to recommend movies of your taste.

This is a project based web app movie recommender, It is developed during the programming task of Precog - IIITD recruitment for membership

This is a kind of minimialistic app in which you have to rate some movies so that we can guess your movie taste and then we will recommend you a movie according to your interest which you can prolly enjoy watching.

Major Points

Know More

Recommends Movies

This is platform where you can just rate few movies and it will show you movies of your taste which you prolly haven't watched yet you won't regret watching them.

Strong Algo

This platform is using one of the most prominent algorithms which is ALS-matrix factoristaion to make the predictions. Thanks to Belle and Koren sir for introducing this greate method.

Nice Interface

It is a ML implementation for predicting the movies which you can see in your web-browser instead of a cool black terminal :).

Technology Stack

Flask

In order to complete this project, I am using Flask and Jinja 2 pages, Flask because it is easy to use for small projects.

Python

Python is one of the most important tool in this project. All ML code and framework is based on python itself.

Database

I have choosen the PostgreSQL for database purposes, two reasons are there mongoDBatlas and mlab is slow in free versions and second is postgreSQL is relational database.

Huge Database

As precribed in the task, instead of using 200 movies database, I have choosen 10K movies with over 1.05 Lacs rating dummies from user data, because when I use 200 movies, even I as a human being is not able to see my results related.

How to use this app

Home Page

The page you are currently looking at is the home page, where yu can get a glimpse of about the app working.

Review Page

The next page when you click any of the "KNOW MORE" button is the review page inn which you can rate a random movie, rate it if its suits your taste else rate it bad, our algorithm will refinr that you dislike this movie, show you better movies.

Final Page

It is the page where we will show you the predicted movies. Try out these movies by searching them on the IMdb websites for details.
